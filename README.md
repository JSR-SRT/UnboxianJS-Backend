# Personal Project

## Overview

A backend server for the UnboxianJS e-commerce web application, built with Node.js, Express.js, and MongoDB.
It provides secure user authentication, efficient API endpoints for product and order management, and a modular architecture for scalability and future integrations.

The backend is built to support real-time data flow between the frontend and database, ensuring reliable operations and a smooth shopping experience.
Future updates will include payment integration and inventory management.

## Tech Stack

Node.js — Runtime environment

Express.js — Web application framework

MongoDB — NoSQL database

Mongoose — ODM for MongoDB

JWT Authentication — Secure user access

Dotenv & CORS — Configuration and cross-origin support

## Features

RESTful API architecture

Secure user authentication (JWT-based)

CRUD operations for user profiles, products, and orders

Error handling and middleware setup
